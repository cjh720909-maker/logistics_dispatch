## 발견한 업무 규칙

* 권역 + 기사명 = 차량
* CB_DRIVER = 실제 작업 단위
* 같은 기사명이라도 권역이 다르면 다른 차량
* Z센터 물량은 배차중량에서 제외
* 동일 P_CODE라도 P_GROUP에 따라 피킹분류가 다를 수 있음
* P_GROUP = 대상 우선 적용
